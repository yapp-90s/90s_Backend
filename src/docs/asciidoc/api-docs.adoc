== Overview

=== HTTP verbs
두가지만 사용합니다. GET! POST!

|===
| Verb | Usage

| `GET`
| GET으로 Request를 보낼때는 따로 보내주실 Parameter가 없습니다.

| `POST`
| Post로 Request를 보낼때는 보통 Json형태로 데이터를 넘겨주시면 됩니다.
|===


=== Status-Code
|===
| Status code | Status Name | Description
| 200
| OK
| 성공!

| 400
| Bad Request
| 무언가 잘못되었다. 하지만 서버 잘못은 아니다!

| 500
| Internal Server Error
| 서버 에러!

| 404
| Not Found
| 없는 주소!
|===
