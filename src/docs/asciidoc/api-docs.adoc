=== HTTP verbs
두가지만 사용합니다. GET! POST!

|===
| Verb | Usage

| `GET`
| GET으로 Request를 보낼때는 따로 보내주실 Parameter가 없습니다.

| `POST`
| Post로 Request를 보낼때는 보통 Json형태로 데이터를 넘겨주시면 됩니다.
|===

==== UPLOAD
----
사진 업로드 CASE는 2개 입니다.

CASE 1 : 필름에 사진 추가
최초 원본사진을 업로드합니다.

CASE 2 : 엘범에 사진 추가
인화된 사진에 스티커를 붙인 사진을 업로드 합니다.
----

==== DOWNLOAD
----
사진 다운로드 CASE는 3개 입니다.

CASE 1 : 원본 사진 ( 최초 사진 )
최초 원본사진을 다운로드합니다.

CASE 2 : 인화된 사진 ( 필름에 추가된 사진 )
필름에 추가 된 뒤 인화된 사진을 다운로드합니다.

CASE 3 : 꾸며진 사진 ( 엘범에 추가된 사진 )
스티커가 부쳐지고 엘범에 된 사진을 다운로드합니다.
----
